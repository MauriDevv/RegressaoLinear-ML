Regressão Linear

Um projeto de machine learning explorando regressão linear na prática, usando Python em um Jupyter Notebook.

A ideia é implementar um modelo de regressão linear do zero (ou quase), entender como ele aprende a relação entre variáveis e visualizar o resultado. Mais um exercício de aprendizado do que algo em produção.

No Notebook


Carregamento e análise exploratória dos dados
Pré-processamento e separação entre treino e teste com train_test_split
Treinamento do modelo com LinearRegression do scikit-learn
Avaliação via métricas
Plot da reta de regressão sobre os dados reais

#O que é regressão linear?

É um dos algoritmos mais simples de ML. Tenta encontrar a melhor reta que descreve a relação entre uma variável de entrada e uma de saída. Simples assim. Mas entender bem esse conceito é a base pra coisas mais complexas.
